Take a screenshot on your Mac

  • ⇧ + ⌘ + 3 — take a screenshot
  • ⇧ + ⌘ + 4 — capture a portion of the screen
  • ⇧ + ⌘ + 4 + ␣ — capture a window or menu
  • ⇧ + ⌘ + 5 — open Screenshot
  • ⇧ + ⌘ + 6 — capture the Touch bar

https://support.apple.com/en-us/HT201361

https://support.apple.com/guide/mac-help/take-screenshots-or-screen-recordings-mh26782/mac

Change your Mac's default screenshot directory

Warning: do not delete the folder you select for screenshots or you're going to have problems.

$ defaults write com.apple.screencapture location /Users/hackerman/Documents/screenshots

If you want to switch things back in future so that images are saved on your desktop again, simply run the command below.

$ defaults write com.apple.screencapture location ~/Desktop
