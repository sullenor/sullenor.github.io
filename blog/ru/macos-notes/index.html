Хоткеи и заметки по операционной системе macOS.

Сделать снимок экрана

  • ⇧ + ⌘ + 3 — снимок всего экрана
  • ⇧ + ⌘ + 4 — снимок части экрана
  • ⇧ + ⌘ + 4 + ␣ — cнимок окна или строки меню
  • ⇧ + ⌘ + 5 — открыть Снимок экрана
  • ⇧ + ⌘ + 6 — снимок панели Touch Bar

Ссылки:

  • https://support.apple.com/en-us/HT201361
  • https://support.apple.com/guide/mac-help/take-screenshots-or-screen-recordings-mh26782/mac

Изменить папку для снимков экрана

Важно: не удаляй папку, в которой хранишь снимки, иначе у тебя будут проблемы.

$ defaults write com.apple.screencapture location /Users/hackerman/Documents/screenshots

По умолчанию macOS сохраняет снимки на рабочий стол. Чтобы вернуть это поведение, выполни команду ниже.

$ defaults write com.apple.screencapture location ~/Desktop
